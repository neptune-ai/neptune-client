Introduction
============

What is Neptune?
----------------

`Neptune <https://neptune.ml/>`_ is a collaboration platform for data science / machine learning (ML) teams that focuses on three areas:

* **Track**: all metrics and outputs in your data science or ML project. It can be model training curves, visualizations, input data, calculated features and so on.
* **Organize**: automatically transform tracked data into a knowledge repository.
* **Collaborate**: share, compare and discuss your work across data science project.

Installation
------------

.. code:: bash

    pip install neptune-client

``neptune-client`` works with Neptune. Make sure to `register here <https://neptune.ml/register>`_, to use its full potential.
